## What I Build at Lumorise

As co-founder, I architect and develop full-stack applications for clients across multiple industries. Here's a snapshot of the production systems I've built:

### Event Ticketing Platform
Full-stack ticketing system for large-scale events with Stripe payments, QR code scanning, guest cart support with automatic merging on sign-in, and a three-tier role system (Customer, Staff, Admin). Built with Next.js, PostgreSQL, Redis, and real-time ticket validation.

### Auto Dealership Platform
Vehicle inventory management system with VIN decoding via NHTSA API, DMS import from CSV via SFTPGo webhooks, vehicle comparison tools, financing calculator, and a secure admin panel with two-factor authentication, trusted device management, and account lockout protection. Built on Next.js, PostgreSQL, and Prisma with Cloudflare Turnstile and a dynamic XML sitemap with JSON-LD structured data.

### Second Dealership Build (Alternate Stack)
A separate car dealership site built on a MySQL backend with hand-rolled JWT authentication and bcrypt hashing instead of Prisma/NextAuth, demonstrating end-to-end delivery across different data layers. Features a status workflow (Draft / Published / Sold), VIN auto-fill, multi-image galleries, and a hidden admin route.

### Client Management Portal
Internal admin portal for managing clients, payments, contracts, and maintenance services. Features Stripe subscription billing, 2FA, rate limiting with Redis, and transactional email via Resend.

### Live Market Pricing Tool
Internal vehicle pricing tool that pulls live market data via MarketCheck API. Auto-decodes VINs, separates clean vs. rebuilt title valuations, and generates dealer-friendly charm pricing recommendations with search bracket optimization.

### CVE Security Scanner
Automated vulnerability scanner for GitHub repositories. Scans transitive dependency trees against npm audit, GitHub Security Advisories, and OSV.dev with zero false positives via proper semver comparison. Runs on cron with Discord webhook alerts and Docker deployment.

### Bilingual Insurance Website
Fully bilingual (English/Spanish) static site with automatic language detection, Google Reviews integration, and mobile-first responsive design. Exported as pure HTML/CSS/JS for fast, low-cost hosting.

### Headless CMS Service Site
React + Vite frontend paired with WordPress as a headless CMS. Uses shadcn/ui components, Radix UI primitives, React Query for data fetching, and server-side form processing via a BitForms PHP proxy.

### Service Business Site with Secure Forms
Marketing and lead-gen site for a security services company, built on Next.js with GSAP motion. Lead forms run through react-hook-form + Zod validation, Upstash Redis rate limiting, and transactional delivery via Resend, with backend-only logic isolated behind server-only modules.

### Tech Agency Website
High-end agency site with an animated deploy-terminal hero (typewriter sequence over a 3D-projected star-field canvas that respects reduced-motion preferences), a dual-row infinite portfolio carousel, and dynamic service pages. Built on Next.js with GSAP-driven motion and Lenis smooth scrolling.
